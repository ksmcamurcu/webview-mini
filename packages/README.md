# WebView Mini - Multi-Platform\n\nSemua platform için uygulamanın paketlenmiş versiyonlarını buradan indirin.\n\n## Platform Desteği\n\n### Windows\n- 📦 `webview-mini-win11-64bit.exe` - Windows 11 64-bit\n- 📦 `webview-mini-win11-32bit.exe` - Windows 11 32-bit\n- 📦 `webview-mini-win10-64bit.exe` - Windows 10 64-bit\n- 📦 `webview-mini-win10-32bit.exe` - Windows 10 32-bit\n\n### macOS\n- 📦 `webview-mini-macos-universal.dmg` - macOS Universal (ARM64 & x86_64)\n- 📦 `webview-mini-macos-intel.dmg` - macOS Intel Only\n\n### Linux\n- 📦 `webview-mini-linux-x86_64.AppImage` - Linux 64-bit AppImage\n- 📦 `webview-mini-linux-x86_64.deb` - Debian/Ubuntu\n- 📦 `webview-mini-linux-x86_64.rpm` - Fedora/RedHat\n\n### Android\n- 📦 `webview-mini-android.apk` - Android APK\n- 📦 `webview-mini-android-universal.apk` - Universal APK (Tüm cihazlar)\n\n### iOS\n- 📦 `webview-mini-ios.ipa` - iOS App\n- 📦 `webview-mini-ios-testflight.ipa` - iOS TestFlight\n\n## Kurulum Rehberi\n\n### Windows\n1. Uygun exe dosyasını indirin (Windows sürümünüze göre)\n2. Dosyaya çift tıklayın\n3. Kurulum tamamlanana kadar bekleyin\n4. Başlat Menüsü'nde \"WebView Mini\" bulun ve çalıştırın\n\n### macOS\n1. DMG dosyasını indirin\n2. Açın ve Applications klasörüne sürükleyin\n3. Applications'dan \"WebView Mini\" başlatın\n4. İlk açılışında \"Açılsın mı?\" onayını verin\n\n### Linux\n- **AppImage**: İndir → Çalıştırılabilir yap → Çalıştır\n- **DEB**: `sudo dpkg -i webview-mini-linux-x86_64.deb`\n- **RPM**: `sudo rpm -i webview-mini-linux-x86_64.rpm`\n\n### Android\n1. APK dosyasını indirin\n2. Dosya Yöneticisi'nde bulun\n3. Dosyaya tıklayın ve kur\n4. Kurulduktan sonra açın\n\n### iOS\n1. IPA dosyasını indirin\n2. iTunes veya Xcode üzerinden kurulumu yapın\n3. Cihazda App Store'dan \"WebView Mini\" bulun ve açın\n\n## Sistem Gereksinimleri\n\n### Windows\n- Windows 10/11 (32-bit veya 64-bit)\n- .NET Framework 4.7+\n- 500 MB depolama alanı\n\n### macOS\n- macOS 11 Big Sur veya daha yeni\n- 300 MB depolama alanı\n- M1/M2 veya Intel işlemci\n\n### Linux\n- Ubuntu 20.04+, Fedora 33+, Debian 11+\n- 400 MB depolama alanı\n- GLIBC 2.31+\n\n### Android\n- Android 10 veya daha yeni\n- 150 MB depolama alanı\n- Device Admin izinleri\n\n### iOS\n- iOS 14 veya daha yeni\n- iPad OS 14 veya daha yeni\n- 200 MB depolama alanı\n\n## Özellikler\n\n✅ Tarayıcı Yönetimi\n✅ Depolama Analizi\n✅ Dijital Saat (Dünya Saatleri)\n✅ Web Erişimi (Opera GX, Ecosia, vb)\n✅ Çevre Dostu Tasarım\n✅ Multi-Language Desteği\n\n## Lisans\n\nMIT License - Özgürce kullanabilirsiniz\n\n## İletişim\n\nHerhangi bir sorun veya önerisi için Issues açın.\n"
