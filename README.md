# WebView Mini

Kullanıcı izni gereklidir ve Google Chrome isteğe bağlı silinebilir.

## Açıklama

WebView Mini, Android cihazlarında tarayıcı uygulamalarını yönetmek ve kaldırmak için tasarlanmış bir uygulamadır. Kullanıcının açık onayıyla:

- Chrome ve diğer tarayıcıları devre dışı bırakabilir
- Depolama alanını görüntüleyebilir
- Performansı iyileştirmek için tarayıcıları yönetebilir
- Yerleşik WebView tarayıcı ile internet erişimi sağlayabilir
- Dünya çapında farklı saat dilimlerinde saati görebilir
- Opera GX ve Ecosia gibi özel tarayıcılara erişebilir

## Özellikler

✅ Tarayıcı yönetimi (devre dışı bırakma/kaldırma)
✅ Depolama analizi
✅ Device Admin izinleri
✅ Yerleşik WebView tarayıcı
✅ Dijital saat - Dünya saatleri
✅ Web Erişimi - Opera GX, Ecosia ve daha fazlası
✅ Kullanıcı dostu arayüz
✅ Multi-Platform Desteği (Windows, macOS, Linux, iOS, Android)

## Sekmeler

1. **Tarayıcılar** - Yüklü tarayıcıları görüntüleyin ve yönetin
2. **Depolama** - Cihazın depolama bilgisini kontrol edin
3. **Web** - Hızlı web erişimi (Opera GX, Ecosia, Google, Bing, DuckDuckGo)
4. **Saat** - Dünya çapında saatleri gerçek zamanlı olarak görüntüleyin
5. **Ayarlar** - Cihaz yöneticisi izinlerini yönetin

- Android 10 veya üzeri

## Kurulum

1. Android Studio'da projeyi açın
2. Gradle senkronizasyonunu yapın
3. APK'yı derleyin ve cihaza kurun
4. Uygulamayı açın ve izinleri onaylayın
